# Viewing Structures

This tutorial covers the basics of viewing atomic structures in ACoord.

## Opening a Structure File

1. Open a supported file in VS Code (`.cif`, `.xyz`, `POSCAR`, etc.)
2. Click the **ACoord Structure Editor** icon in the editor title bar
3. Or right-click → **Reopen with → ACoord Structure Editor**

## Navigation Controls

| Action | Control |
|--------|---------|
| Rotate | Left-click and drag |
| Pan | Right-click and drag |
| Zoom | Scroll wheel |
| Reset | Click Reset button in toolbar |

## Display Settings

### Atom Size

1. Open the **Size & Style** panel
2. Use the **Atoms radius** slider to adjust selected atoms
3. Click **Reset to element default** or **Set to covalent radius** for defaults

### Bond Thickness

1. Open the **Size & Style** panel
2. Use the **Bond thickness** slider

### Color Scheme

1. Open the **Colors** panel
2. Select a color scheme from the dropdown
3. Select atoms in the 3D view
4. Click **Apply** in the Brush panel (left toolbar)

### Lighting

1. Open the **Lighting** panel
2. Adjust Ambient, Key, Fill, or Rim lights

## Selection

### Select Single Atom

- Click directly on an atom

### Multi-Select

- Hold **Ctrl/Cmd** and click multiple atoms

### Box Select

1. Ensure **Select** tool is active (press `V`)
2. Click and drag in empty space
3. Release to select atoms inside box
4. Hold **Ctrl/Cmd** while releasing to add to selection
5. Hold **Alt** while releasing to subtract from selection

### Deselect

- Click in empty space or press **Esc**

## Measuring Bonds

1. Select two atoms (Ctrl/Cmd+click both)
2. Distance displays in the Properties panel
